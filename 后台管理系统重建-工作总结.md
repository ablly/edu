## 📋 核心设计特点

### 设计风格
- **主题**：AI教育 + 科幻未来风格
- **色调**：Dark Mode（深空背景）
- **主色**：霓虹蓝 #00D4FF
- **辅色**：科幻紫 #B15BFF
- **特效**：玻璃态、发光效果、扫描线动画

### 技术栈
- **前端**：React 18 + TypeScript + Ant Design 5
- **动画**：Framer Motion
- **图表**：ECharts + React Sparklines
- **状态**：Zustand + React Query
- **后端**：Flask + PostgreSQL（已有）

### 核心模块（精简版，5个）

1. **数据总览**：实时统计、趋势图表、系统监控
2. **用户管理**：14项完整功能（列表、筛选、编辑、批量操作等）
3. **会员管理**：套餐管理、早鸟优惠、会员分析
4. **订单管理**：订单查询、退款处理、收入统计
5. **支付记录**：支付查询、对账报表、异常监控

**移除的模块**（不在管理后台范围）：
- ❌ 学生管理（属于教学业务）
- ❌ 作业管理（属于教学业务）
- ❌ 题库管理（属于教学业务）
