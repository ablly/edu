## 🚀 下一步操作

### 立即开始开发

```bash
# 1. 创建React项目
npm create vite@latest admin-frontend -- --template react-ts

# 2. 进入目录
cd admin-frontend

# 3. 安装依赖
npm install
npm install antd @ant-design/icons framer-motion echarts echarts-for-react react-countup @tanstack/react-query axios dayjs zustand

# 4. 启动开发服务器
npm run dev
```
